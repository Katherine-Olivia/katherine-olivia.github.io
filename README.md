# Katherine Olivia - Personal Portfolio Website

## Overview

This is a responsive and fully accessible portfolio website built using **HTML**, **CSS**, and **JavaScript**. The website showcases the professional profile of **Katherine Olivia**, including experience, projects, technical skills, education, and certifications. It is designed for academic, internship, and professional networking purposes.

## Features

- Clean and modern responsive design
- Smooth scroll navigation with fixed navbar
- Typewriter effect introduction
- Downloadable resume
- Section-based layout: About, Experience, Skills, Projects, Education, Certifications, and Contact
- Scroll-to-top button
- Hover effects and scroll-based animations
- Separate icon support for GitHub, LinkedIn, Email, and more

## Technologies Used

| Layer | Tools / Frameworks |
|-------|--------------------|
| **Frontend** | HTML5, CSS3, JavaScript |
| **Fonts & Icons** | Google Fonts (`Courier Prime`), Custom SVG Icons |
| **Effects** | Typewriter JS logic, Scroll-triggered animations |
| **Hosting Compatibility** | GitHub Pages, Vercel, Netlify, or any static hosting |

## Setup Instructions

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Katherine-Olivia/portfolio-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-website
   ```
3. Open `index.html` in your preferred browser.

> _Note: Ensure image paths and resume file are present in the root or properly referenced._
